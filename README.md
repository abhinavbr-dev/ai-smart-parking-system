# Real-Time Smart Parking System Using AI, Cloud Computing and IoT

## Overview
The **Real-Time Smart Parking System Using AI, Cloud Computing and IoT** is an intelligent parking management solution designed to reduce parking congestion and improve parking efficiency through real-time monitoring and smart slot recommendations.

The system integrates:
- IoT sensors for vehicle detection
- ESP32 microcontroller for data transmission
- Firebase cloud platform for real-time synchronization
- AI-based logic for smart parking recommendations
- Web application for user interaction and parking management

This project demonstrates the practical implementation of Smart City technologies using IoT, Cloud Computing, and Artificial Intelligence.

---

# Features

- Real-time parking slot monitoring
- Occupied/Vacant slot detection
- AI-based parking slot recommendation
- Cloud database integration using Firebase
- Real-time frontend updates
- Parking slot booking
- Demo payment module
- Responsive web interface
- IoT sensor integration using ESP32
